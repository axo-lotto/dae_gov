## üîë Key Design Features

### **1. Whiteheadian Eternal Objects**
- Templates = pure potentials (eternal objects)
- Ingression = selection based on felt states (meta-atoms, zones, organ coherences)
- Concrescence = blending templates into unified emission
- Objective immortality = successful templates inform future selections

### **2. Trauma-Informed Gating**
- **NEVER** inject humor/small talk in Zone 4/5
- **NEVER** use personality layer when NDAM > 0.7 (crisis)
- **ALWAYS** check user preferences (humor_tolerance, small_talk_openness)
- **ALWAYS** preserve therapeutic core over personality

### **3. Template Variables**
- Brackets `{variable}` indicate dynamic content
- Examples: `{theme}`, `{past_conversation}`, `{inside_joke}`, `{organ}`, `{zone}`
- PersonaLayer will substitute with actual conversation data

### **4. Progressive Complexity**
- **Phase 1 (now):** Discrete template selection
- **Phase 2 (months 1-2):** Family-specific preferences learned
- **Phase 3 (months 2-4):** Template composition & blending
- **Phase 4 (months 4-12+):** Generative emission via semantic manifold

### **5. LLM-Optional Architecture**
- Templates work standalone (zero dependency)
- LLM enhances factual/creative (optional)
- **NEVER** replaces therapeutic core
- Graceful fallback if LLM unavailable
