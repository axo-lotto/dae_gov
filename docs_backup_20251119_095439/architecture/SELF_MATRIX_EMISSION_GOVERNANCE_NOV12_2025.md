## ğŸ§¬ Coherent Attractors: Validated Lures from Corpus

### What Are Coherent Attractors?

**Whiteheadian Definition**: Lures for feeling that draw actual occasions toward coherence through felt propositions.

**Therapeutic Translation**: Validated phrases/patterns that:
- **Respect** the current SELF-distance zone
- **Invite** movement toward Core SELF (without force)
- **Embody** trauma-informed principles (safety, choice, collaboration)
- **Emerge** from validated corpus (van der Kolk, IFS, Wordsworth, existing training pairs)

**Architecture Pattern**:
```
Coherent Attractor = {
    zone: SELF matrix zone (5 zones),
    lure_type: inquiry | reflection | grounding | witnessing | integration,
    phrase_pattern: compositional template,
    validated_source: corpus reference,
    polyvagal_alignment: ventral | sympathetic | dorsal,
    window_of_tolerance: safe | edge | breach,
    therapeutic_move: somatic | relational | cognitive | integrative
}
```
